🚀 Welcome to MHD++ Framework

Version 1.0.0

A modern, lightweight C++ web framework for building high-performance web applications with elegant simplicity.

✨ Features

🔧 Pure C++ Implementation

Built entirely in modern C++ for maximum performance and minimal dependencies.

🌐 Dynamic Routing

Flexible routing system with support for dynamic URLs and parameter extraction.

📂 Static File Serving

Efficient static file serving with automatic MIME type detection and caching.

📄 Template Engine

Built-in HTML template rendering system for dynamic content generation.

🧩 Modular Architecture

Clean, modular design that's easy to extend and customize for your needs.

⚡ High Performance

Optimized for speed with minimal overhead and efficient memory usage.

🗄️ Built-in ORM

Simple Object-Relational Mapping with SQLite support using C++ OOP principles.

🔍 Query Builder

Object-oriented approach to building database queries with clear C++ syntax.

🗄️ Object-Relational Mapping (ORM)

🗃️ SQLite Integration

Lightweight, file-based database with zero-configuration setup and deployment.

🏗️ C++ OOP Models

Define your data models using standard C++ classes with clear object-oriented design.

🔍 Simple Queries

Clean C++ interface for common database operations like CRUD and basic filtering.

📊 Basic Relations

Support for fundamental relationships between your C++ model objects.
