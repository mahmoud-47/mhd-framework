🚀 Welcome to MHD++ Framework

Version 1.1 - Now with Session Support 🔑

A modern, lightweight C++ web framework for building high-performance web applications with elegant simplicity.

🛡️ Security Enhancements

🔐 Session Management

Lightweight `key=value` session system to persist state across requests using secure identifiers.

🧬 CSRF Protection

All form submissions include cryptographic tokens to guard against Cross-Site Request Forgery attacks.

🛡️ SQL Injection Defense

Database queries use parameterized execution through a C++ Query Builder to prevent injection vectors.

⏰ MhdDateTime Support

Custom datetime utility for managing session timeouts, user activity tracking, and token expiration.

📁 Secure File Serving

Using MhdFile to expose internal headers and static files safely with path validation.

✨ Features

🔧 Pure C++ Implementation

Built entirely in modern C++ for maximum performance and minimal dependencies.

🌐 Dynamic Routing

Flexible routing system with support for dynamic URLs and parameter extraction.

📂 Static File Serving

Efficient static file serving with automatic MIME type detection and caching.

📄 Template Engine

Built-in HTML template rendering system for dynamic content generation.

🧩 Modular Architecture

Clean, modular design that's easy to extend and customize for your needs.

⚡ High Performance

Optimized for speed with minimal overhead and efficient memory usage.

🗄️ Built-in ORM

Simple Object-Relational Mapping with SQLite support using C++ OOP principles.

🔍 Query Builder

Object-oriented approach to building database queries with clear C++ syntax.

🧠 MhdDateTime Integration

Custom C++ class to manage time-sensitive logic like session control and timestamps.

📁 MhdFile Support

Retrieve and serve C++ headers securely and efficiently using the MhdFile handler.
